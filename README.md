# Solar System Simulator

Симулятор Солнечной системы.
Учебный проект студентов 2 курса группы ЛПб-ИСиТ-24-1 Лебедевой Е.Н. и Гавриловой А.Р.

h1 Установка
Требования
Python 3.8 или выше
Pygame 2.5+


h1 Управление
Клавиатура
Стрелки вверх/вниз   Изменить скорость симуляции

+ / -  Увеличить/уменьшить масштаб

R Сброс вида

E Следовать за Землей

J  Следовать за Юпитером

T   Следовать за Сатурном

Пробел   Пауза/продолжить

Мышь
Колесико - Изменить масштаб

ЛКМ - Нажать на кнопки интерфейса


h2 Скорость симуляции
Фиксированные значения: 1x, 2x, 5x, 10x, 20x, 30x, 40x, 50x

h2 Масштаб
Фиксированные значения: 0.5x, 1x, 2x, 5x, 8x

h2 Настройка
Параметры планет можно изменить в файле settings.json


h1 Структура проекта
